# HH-бот — парсер вакансий hh.ru с авто-откликом

Программа ищет вакансии на hh.ru по вашим критериям и **сама откликается** на подходящие.
Работает не через API, а управляя настоящим браузером (Playwright): вы один раз
логинитесь вручную, дальше бот кликает за вас. Есть графический интерфейс.

## Установка

```bash
pip install -r requirements.txt
playwright install chromium
```

## Запуск

```bash
python main.py
```

## Как пользоваться

1. Нажмите **«Войти»** — откроется окно браузера, войдите на hh.ru вручную
   (логин/пароль, при необходимости капча и SMS). Сессия сохранится — в следующий
   раз вход не потребуется.
2. Заполните **критерии** (профессии, регион, зарплата, исключающие слова,
   сопроводительное письмо, дневной лимит) и нажмите **«Сохранить критерии»**.
3. **«Найти»** — предпросмотр: бот соберёт вакансии в таблицу, ни на что не откликаясь.
4. **«Запустить отклики»** — бот сам откликнется на все подходящие вакансии
   с паузами и в пределах дневного лимита. **«Стоп»** — прервать.

## Настройки (`config.yaml`)

| Поле | Значение |
|------|----------|
| `professions` | список поисковых запросов (профессий) |
| `region` | id региона hh.ru (1 = Москва, 2 = СПб, 113 = Россия) |
| `salary_min` | минимальная зарплата, 0 = не фильтровать |
| `exclude_words` / `include_words` | слова-стопы и обязательные слова в названии |
| `cover_letter` | текст сопроводительного письма |
| `daily_limit` | дневной лимит откликов (лимит hh.ru ~200) |
| `delay_seconds` | пауза между откликами `[мин, макс]` |
| `max_pages` | сколько страниц выдачи обходить на запрос |

## Важно

- **Лимиты и блокировки.** hh.ru ограничивает число откликов в день (~200) и может
  блокировать за подозрительную активность. Не снижайте паузы и держите лимит ниже 200.
- **Капча.** Если при отклике появляется капча, бот ставит вакансию в статус «ошибка»
  с пометкой — пройдите капчу вручную в окне браузера и запустите снова.
- **Хрупкость.** Вёрстка hh.ru меняется. Если перестало находить элементы — правьте
  селекторы в `hh_bot/selectors.py`.
- Используйте ответственно и в рамках правил сервиса hh.ru.

## Структура

```
main.py                 точка входа (GUI)
config.yaml             ваши критерии
hh_bot/
  config.py             загрузка/сохранение критериев
  models.py             модель вакансии
  browser.py            Playwright: сессия и логин
  selectors.py          все селекторы hh.ru
  search.py             построение URL и парсинг выдачи
  filters.py            отбор по критериям
  applier.py            авто-отклик
  storage.py            SQLite-история (дедупликация)
  worker.py             рабочий поток (владеет браузером)
gui/app.py              интерфейс Tkinter
```
